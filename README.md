# some-random-shell
Random Windows 10 Shell with some cool functions

## Setup
- Clone this Repo:
  - Using git: `git clone https://github.com/Stift007/some-random-shell.git && pip install pywhatkit phonenumbers`
  - Direct Download: [here](https://www.mediafire.com/file/k652o3e7i22yb1r/bash.exe/file)
  - From Source (wget): `wget https://pastebin.com/raw/kqvzSfTL && pip install pywhatkit phonenumbers`
  - From Source (graw): `graw get https://pastebin.com/raw/kqvzSfTL && pip install pywhatkit phonenumbers`

## Launch
- If downloaded from Source, run `python3 bash.py`
- If downloaded as .exe, just run bash.exe
